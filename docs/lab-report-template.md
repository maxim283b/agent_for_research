# Отчет по лабораторной работе 3: Агентный искусственный интеллект

## 1. Постановка задачи

Опишите задачу научно-информационного поиска и структуру итогового обзора:

- определение темы;
- основные подходы;
- 3-5 ключевых работ;
- применения;
- ограничения;
- использованные источники.

## 2. Архитектура решения

Опишите:

- используемую модель;
- режим `baseline`;
- режим `agent`;
- режим `agent + evaluator`;
- инструменты поиска и извлечения данных;
- структуру `AgentState` и формат `trace`.

## 3. Набор тем и схема экспериментов

Зафиксируйте:

- восемь тем;
- три основные конфигурации;
- sweep по `topK = 3, 5, 8`;
- sweep по `maxSteps = 4, 6, 8`.

## 4. Результаты

Вставьте таблицы:

- сравнение `baseline / agent / agent + evaluator`;
- влияние `topK`;
- влияние `maxSteps`.

Добавьте графики по:

- latency;
- числу шагов;
- completeness;
- rubric.

## 5. Разбор неудачных кейсов

Выберите минимум три слабых запуска и разберите:

- какой шаг дал основную деградацию;
- были ли лишние шаги;
- насколько ответ опирался на реальные источники;
- какие ограничения проявились у агента.

## 6. Итоговый вывод

Ответьте:

- какая конфигурация лучшая;
- за счет чего она выигрывает;
- каков компромисс между качеством и latency;
- в каких условиях агентный подход оправдан.
